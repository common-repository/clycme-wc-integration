== Description ==
This plugin allows you to integrate your WooCommerce store with clyc.me for making donations based on your products sales.
You only can use this plugin if you have an active plan on https://app.clyc.me.
On https://app.clyc.me you will find the "Customer" page on the menu, you can create your API KEY in "My Integrations" menu. You should put it on your store config, otherwise, the calls made by this plugin will do not do anything.
